# FoodBridge – Connecting Surplus Food with Hungry Hearts

FoodBridge is a mobile application designed to reduce food waste and fight hunger in Bangladesh by connecting food donors, volunteers, NGOs, and buyers.

## Features
- Food donation & resale
- Volunteer pickup & delivery
- Real-time tracking
- Role-based authentication
- Firebase backend
- Flutter mobile app

## Tech Stack
- Flutter
- Firebase Authentication
- Firestore
- Firebase Cloud Storage
- Firebase Cloud Messaging
